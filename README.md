# Heroes CRUD API

Proyecto full stack desarrollado con Flask, MySQL, HTML, CSS y JavaScript para gestionar héroes mediante operaciones CRUD.

## Funcionalidades
- Listar héroes
- Crear héroes
- Editar héroes
- Eliminar héroes

## Tecnologías
- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Estructura
- Backend: API REST con Flask
- Frontend: interfaz en HTML/CSS/JS
- Base de datos: MySQL

## Cómo ejecutar
1. Crear la base de datos con el archivo SQL
2. Configurar credenciales de MySQL
3. Ejecutar `app.py`
4. Abrir el frontend en el navegador
